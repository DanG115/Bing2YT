Bing2YT Changelog

Your current version:

v5.0.0 - 07/02/2026

  • + Added a new redirect system using an intermediate page.
  • Improved the settings UI, now available as a toggleable beta.
  • Fully redesigned core systems with a new modular architecture.
  • + Added updated extension screenshots.
  • + Added new UI settings to switch the search button between icon and full button.
  • Dark mode is now fully implemented across all pages.
  • The welcome page has been reworked.
  • − Removed font and text size customisation (may return in a future update).
  • Fixed the “Search on YouTube” button not opening correctly or in a new tab.
  • − Disabled manual update checks.
  • Redirect toast messages no longer accept parameters.

v4.0.4 - 12-12-2025

  • Fixed redirect not triggering on certain Bing video pages.
  • Settings page UI reworked (dark theme).
  • Improved compatibility with the latest Bing UI changes.
  • Changelog page now opens automatically on extension update.
  • Welcome page now opens automatically on extension install.
  • Minor performance optimisations and bug fixes.

v4.0.3 - 28-11-2025

  • Enhanced YouTube redirect detection for more reliable performance.
  • Added video info tooltips for Bing video tiles.
  • Fixed settings saving issue for some users.
  • Introduced a bug reporting system via forms.
  • Redesigned the YouTube search button and UI.
  • Redirect system now retries multiple times if the video URL isn't immediately found.
